# Travel planning assistant based on large models

#### Description
The "14th Five-Year Tourism Development Plan" states: "We should strengthen independent innovation, integrate advantageous resources, and accelerate the advancement of smart tourism characterized by digitalization, networking, and intelligence." Against this backdrop, the importance of intelligent tourism recommendations is becoming increasingly prominent. However, current itinerary recommendation software faces issues such as insufficient personalization, automation, and intelligence. This project aims to develop a personalized travel planning assistant based on technologies like large-scale models to enhance the tourist experience.
